// configuração firebase

Para o chat funcionar entre vários computadores, você precisa de um banco de dados Firebase gratuito.

1. Acesse console.firebase.google.com
2. Crie um projeto → Realtime Database → Iniciar no modo de teste
3. Em Configurações do projeto → Seus apps → Web → copie o objeto firebaseConfig e cole abaixo.
